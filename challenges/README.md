# Desafios de Lógica de Programação - Curso JavaScript do Zero!

## Instruções
- Cada arquivo tem **lacunas** (`// LACUNA:`) para você preencher.
- Execute com `node challenges/XX-nome.js` e verifique o output esperado.
- Dificuldade progressiva: variáveis → condicionais → loops → arrays → funções → FizzBuzz.

1. **01-variaveis.js**: Operações básicas → output 15
2. **02-condicionais.js**: Par/ímpar → classifique [1,2,3,4]
3. **03-loops.js**: Soma 1-10 → 55
4. **04-arrays.js**: Filtre pares → [2,4,6]
5. **05-funcoes.js**: Verifique primo → true/false para 7,10
6. **06-fizzbuzz.js**: FizzBuzz 1-20
